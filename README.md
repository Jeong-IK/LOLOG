## 기능
1. 롤 아이디를 입력 받아 이용자의 전적을 조회
2. 사용 빈도가 높은 순서대로 정렬 및 출력
 - 사용 빈도를 백분율로 계산
 - 1~3위까지 이미지 출력 4위 이하는 이름만 출력
3. 승률이 높은 순서대로 정렬 및 출력
 - 마찬가지로 백분율로 계산
 - 전적을 출력
 - 가장 최근부터 차례대로 승리한 전적 출력
